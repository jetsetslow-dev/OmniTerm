# AI-Assisted Development

OmniTerm has been developed with assistance from AI coding tools, with human
review, testing, and release approval before publication.

Project policy:

- Do not paste third-party proprietary code into the project through AI tools.
- Prefer standard platform APIs, original implementation, and properly licensed
  dependencies.
- Keep dependency licenses documented in `THIRD_PARTY_NOTICES.md`.
- Review generated changes for correctness, security, privacy, and licensing
  before release.

There is no special open-source license restriction solely because AI tools were
used. The project license and dependency licenses still control use of the code.
